### Description

WooCommerce Accessibility Auto-Fixer is the developer-friendly alternative to overlay tools like AccessiBe, UserWay, and AudioEye. Instead of placing a JavaScript widget over broken HTML, this plugin actually fixes your source code.

**Why it matters:**
- Overlays don't change your HTML — WAVE and axe DevTools still detect violations
- Courts have rejected overlays as evidence of ADA compliance (Murphy v. Eyebobs LLC)
- This plugin fixes actual code-level accessibility issues

**What it fixes (automated, one-click):**
- Missing alt text on product images
- Low color contrast in buttons and text
- Missing form labels in checkout
- Invisible keyboard focus indicators
- Missing skip links
- Keyboard traps in product variant modals
- Incorrect heading hierarchy
- Missing ARIA landmarks
- Missing inline error messages

**Plans:**
- Free: 10 fixes/month (skip link, focus indicator, product alt text)
- Pro: $19/month — unlimited fixes across all 9 categories

**Requirements:**
- WordPress 5.8+
- WooCommerce 7.0+
- PHP 7.4+
