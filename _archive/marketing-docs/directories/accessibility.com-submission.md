### Description (500 char limit)

AmazingPlugins builds code-level accessibility fixers for e-commerce platforms. Our WooCommerce Accessibility Auto-Fixer and Shopify Accessibility Auto-Fixer automatically detect and fix WCAG 2.1 AA violations — missing alt text, keyboard traps, form labels, color contrast, focus indicators, heading hierarchy, and ARIA landmarks.

Unlike overlay tools (AccessiBe, UserWay), our plugins fix issues at the source code level. Courts have rejected overlays as compliance evidence; our approach is defensible.

Free tier available. Plans from $19/mo. Supports WooCommerce 7.0+, Shopify, WordPress 5.8+.
