## Description (to use on SaaSHub)

WooCommerce Accessibility Auto-Fixer is the code-level alternative to overlay widgets for WooCommerce stores. It automatically detects and fixes the 9 most common WCAG 2.1 AA violations directly in your WooCommerce templates — no overlay, no JavaScript injection, no legal risk.

Unlike AI overlay tools (AccessiBe, UserWay, AudioEye), this plugin modifies actual HTML/CSS source. Courts have accepted code-level fixes as compliance evidence; overlay-only compliance has been rejected.

**What it fixes (free tier):**
- Missing alt text on product images
- Missing focus indicators
- Missing skip navigation links

**What it fixes (Pro — $19/mo):**
- Keyboard navigation traps in product variant pickers
- Checkout form labels (for/id association)
- Screen reader error announcements (aria-live)
- Color contrast on CTAs and pricing
- Heading hierarchy on product/archive pages
- ARIA landmark roles

**Why not an overlay:**
- Fixes source code, not just visual appearance
- Specifically built for WooCommerce
- Court-defensible compliance
- No performance penalty from JS injection

**Trial:** Free tier works indefinitely
