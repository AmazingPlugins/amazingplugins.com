### Tagline / Short Description

### Full Description

WooCommerce Accessibility Auto-Fixer is a WordPress plugin that automatically detects and fixes the most common WCAG 2.1 AA violations in WooCommerce online stores.

**What it fixes (free tier — always free):**
- Missing alt text on product images
- Missing focus indicators
- Missing skip links

**What it fixes (Pro — $19/month):**
- Keyboard trap in product variant pickers
- Checkout form labels missing or disassociated
- Error messages not announced via ARIA live regions
- Color contrast issues on buttons and body text
- Heading hierarchy breaks
- Missing ARIA landmark roles

**Why this matters:**
Under the ADA, European Accessibility Act (EAA), and WCAG 2.1 AA standards, e-commerce stores are legally required to be accessible. Courts have rejected overlay widgets (AccessiBe, UserWay, AudioEye) as evidence of compliance — code-level fixes are the defensible path.

Unlike overlays that sit on top of broken code, this plugin fixes the actual WooCommerce template output at source.

**Requirements:**
- WordPress 5.8+
- WooCommerce 6.0+
- PHP 7.4+
