## PR Description

```markdown
## Add AmazingPlugins Shopify Accessibility Fixer

This PR adds the AmazingPlugins Shopify Accessibility Fixer to the Shopify apps section.

### Why this belongs

Accessibility is a growing concern for Shopify store owners, especially with increasing ADA/EAA enforcement. Unlike overlay tools (AccessiBe, UserWay) that add JavaScript widgets on top of the page, AmazingPlugins fixes WCAG 2.1 AA violations at the source code level — which courts and accessibility auditors actually recognize.

### What it fixes

- Missing alt text on product images
- Keyboard navigation traps
- Missing form labels on checkout/account forms
- Low color contrast ratios
- Missing focus indicators
- Incorrect heading hierarchy
- Missing ARIA landmarks
- Missing skip links

### Source

GitHub: https://github.com/AmazingPlugins/shopify-accessibility-fixer
```
