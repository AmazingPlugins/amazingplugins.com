## AlternativeTo Listing Content

### Description

WooCommerce Accessibility Auto-Fixer is the code-level alternative to overlay widgets like AccessiBe, UserWay, and AudioEye. Unlike overlays that sit on top of broken HTML, this plugin fixes WCAG 2.1 AA violations directly in your WooCommerce templates.

**Key differences from overlay products:**
- Fixes actual HTML/CSS/JS source code, not just visual appearances
- Specifically built for WooCommerce (not a generic overlay)
- Court-defensible: unlike overlays, code-level fixes have been accepted as compliance evidence
- Free tier available (overlays are always expensive)

**What it fixes:**
- Missing alt text on product images
- Missing form labels in checkout
- Keyboard navigation traps
- Missing focus indicators
- Color contrast issues
- Poor heading hierarchy
- Missing ARIA landmarks

**Alternatives considered:**
- AccessiBe: AI overlay, rejected by courts as compliance evidence
- UserWay: Same overlay problem as AccessiBe
- AudioEye: Real accessibility team, but targets enterprise budgets ($500+/mo)
- Equalize Digital: Reporting tool only, no actual fixes
