## Free Keyword Research Tools

### 1. Ubersuggest (Neil Patel) — ube.rs / neilpatel.com/ubersuggest
**Cost:** Free tier (daily limit), $29/mo full
**Best for:** Keyword ideas, search volume, SEO difficulty, competitor keywords
**How to use:** Enter a seed keyword, get 100+ keyword ideas with volume + difficulty
**Daily limit:** ~3 searches without logging in, more with free account

### 2. Keyword Surfer — Chrome Extension
**Cost:** 100% free
**Best for:** See search volume + CPC directly in Google search results
**How to use:** Install Chrome extension, search Google normally, see volume next to each result
**Limitation:** Only shows keywords Google suggests or shows in results

### 3. Keywords Everywhere — Chrome Extension ($21/yr)
**Cost:** $21/year (cheap)
**Best for:** Search volume + CPC + competition in Google, YouTube, Amazon search
**How to use:** Install extension, buy API key, see data next to every search result
**Advantage:** Works across Google, Amazon, YouTube, Bing simultaneously

### 4. AnswerThePublic — answerthepublic.com
**Cost:** Free (limited), $99/mo Pro
**Best for:** Question-based long-tail keywords ("how to...", "why does...", "what is...")
**How to use:** Enter a topic, get a visual map of question keywords people search
**Free limit:** 2 searches/day

### 5. Answer Socrates — answersocrates.com
**Cost:** Free
**Best for:** Simple question keyword discovery
**How to use:** Enter a keyword, get variations of question-based searches

### 6. Google Keyword Planner — ads.google.com (free account required)
**Cost:** Free (requires Google Ads account)
**Best for:** Real search volume data from Google, keyword ideas from seed
**How to use:** Create free Google Ads account → Tools → Keyword Planner
**Advantage:** Most accurate search volume data (from Google itself)
**Limitation:** Designed for PPC, shows ranges for low-volume keywords

### 7. Google Search Console — search.google.com/search-console
**Cost:** Free
**Best for:** See what keywords YOUR site already ranks for
**How to use:** Submit sitemap, check Performance report after traffic arrives
**Best for:** Finding long-tail keywords you're already winning on

### 8. AlsoAsked — alsoasked.com
**Cost:** Free tier (3 searches/day), $29/mo
**Best for:** "People also ask" data — question clusters around a topic
**How to use:** Enter a keyword, get related questions grouped by intent

### 9. ChatGPT / Claude (for brainstorming)
**Cost:** Free/PRO
**Best for:** Generating keyword clusters, content angles, related topics
**How to use:** "Give me 20 long-tail keywords for [topic] targeting [audience]"
**Limitation:** No search volume data, requires human judgment

**Easy (low competition, quick wins):**
- Keywords with "woocommerce" + "accessibility" + specific issue (focus indicator, landmark, skip link)
- Long-tail questions ("how to fix woocommerce alt text missing")
- Brand-adjacent ("woocommerce [brand] alternative")

**Medium (worth targeting):**
- woocommerce bulk edit plugin
- woocommerce accessibility plugin
- woocommerce ada compliance
- woocommerce bulk edit variations

**Hard (high competition, later):**
- woocommerce accessibility lawsuit
- best woocommerce bulk edit plugin
- wordpress accessibility plugin
